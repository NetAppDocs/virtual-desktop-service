== Zoom for VDI/WVD overview
This script package installs/uninstalls _Zoom VDI-WVD_ using PowerShell to do the deployment.

NOTE: Zoom performance is improved if audio redirection is also enabled for the VDI/WVD environment.

=== Default shortcut path
The default shortcut path will be entered below, for this application the shortcut is: `\\shortcuts\Zoom VDI.lnk`
