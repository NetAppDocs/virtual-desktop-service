== Microsoft Edge Chromium overview
This script package installs/uninstalls _Microsoft Edge Chromium_ using the Chocolatey package manager (https://chocolatey.org/) to do the deployment. Chocolatey is deployed by VDS when VMs are created but this script will also check and install Chocolatey as a prerequisite if it is missing.

=== Default shortcut path
The default shortcut path will be entered below, for this application the shortcut is: `\\shortcuts\Microsoft Edge.lnk`
